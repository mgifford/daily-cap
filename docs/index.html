About

Daily CAP benchmarks the quality and accessibility of Canadian government websites using automated scanning:

  • Daily automated scans with Lighthouse and ScanGov
  • Bilingual (English/French) analysis and parity detection
  • Performance, accessibility, and SEO metrics
  • Service-pattern and impact analysis
  • Canadian-focused data sources (recent activity, top tasks, curated endpoints)
